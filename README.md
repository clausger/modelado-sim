# Modelado y Simulacion - UADE

Aplicacion Streamlit para metodos numericos.

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Modulos

| Modulo | Estado |
|--------|--------|
| Raices y Punto Fijo | WIP |
| Newton-Raphson / Aitken | WIP |
| Interpolacion y Derivacion | WIP |
| Integracion Numerica | WIP |
| **Montecarlo** | **Completo** |
| EDOs | WIP |

## Stack

- Python + Streamlit
- Plotly (graficos interactivos)
- NumPy + SciPy (calculo numerico)
- SymPy (calculo simbolico)
